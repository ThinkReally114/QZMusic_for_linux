# QZ Music for Linux

一款简洁美观的跨平台音乐播放器，提供优雅的桌面音乐体验
安卓版本请访问：[官方网站](https://music.qz.shiqianjiang.cn)

## 📖 项目说明

本项目为 **Vue + Electron** 的学习实践作品，旨在完善QZ Music的多平台生态。
Linux版本依赖MPV音乐播放器

> ⚠️ 注意：
> - QZPlugin 仅提供插件运行框架，不包含任何默认音乐插件
> - AMLL 提供功能接口，不关联具体音乐数据源
